# 道教疏文排版系統 (網頁版)

一個強大且彈性的網頁版疏文排版工具，專為道教宮廟與法事設計。這是一個純前端的解決方案，無需安裝額外軟體，即可在瀏覽器中完成高品質的疏文排版與列印。

## 🌟 核心特色

-   **靈活排版控制**：支援直式（由右至左）與橫式排列，可自由拖曳區塊、調整字型大小、顏色與行高。
-   **批次自動套印**：支援匯入 CSV 檔案，透過 `{{欄位名稱}}` 變數自動生成多份疏文，大幅節省重複作業時間。
-   **本機信眾名冊**：內建名冊管理系統，資料儲存於瀏覽器本機 (IndexedDB)，支援 CRUD 操作與快速搜尋，隱私有保障。
-   **範本市場與管理**：提供官方預設範本，並支援匯入/匯出 JSON 格式的專案檔，方便重複使用與分享。
-   **摺紙與印章輔助**：內建摺紙輔助線功能（自訂摺數）與電子印章生成工具。
-   **農曆工具整合**：自動換算今日農曆日期，並可快速插入疏文格式。
-   **法師讀稿模式**：專為法師設計的「大字報/讀稿機」模式，適合法會現場使用。

## 🚀 快速開始

這是一個綠色版單頁面應用程式，無需任何伺服器環境即可運行。

1.  下載本專案原始碼。
2.  直接使用瀏覽器（推薦 Chrome 或 Edge）開啟 `index.html`。
3.  即可開始設計您的疏文。

## 🛠️ 技術棧

-   **核心框架**：[React](https://reactjs.org/) (Standalone via Babel)
-   **樣式系統**：[Tailwind CSS](https://tailwindcss.com/)
-   **資料儲存**：[localforage](https://github.com/localForage/localForage) (IndexedDB 管理)
-   **CSV 解析**：[PapaParse](https://www.papaparse.com/)
-   **農曆計算**：[lunar-javascript](https://github.com/6tail/lunar-javascript)
-   **PDF 匯出**：[html2pdf.js](https://github.com/eKoopmans/html2pdf.js)
