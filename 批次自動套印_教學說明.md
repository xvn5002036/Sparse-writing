# 道教疏文排版系統 (高級版) - 批次自動套印使用說明

這項「批次自動套印」功能是本系統的核心商業價值所在，能夠讓宮廟在處理大型法會（如中元普渡、安太歲）時，省去大量的謄寫與複製貼上時間。只需要準備好 Excel 名單，系統就能一次幫您產出數百份排版完美的疏文！

以下是詳細的操作步驟：

## 第一步：準備信眾名單 (CSV 格式)

1. 打開您的 Excel 或 Google 試算表。
2. 在第一列（標題列）打上您需要的資料欄位名稱，例如：「姓名」、「地址」、「祈求事項」。
3. 在下方依序填入信眾的資料。
   * **注意：資料中不要有不必要的特殊符號或換行。**
4. 點選「檔案」 -> 「另存新檔」，將檔案格式選擇為 **CSV (逗號分格)**。

> **Excel 範例格式**：
| 姓名 | 地址 |
| :--- | :--- |
| 王小明 | 台北市信義區OO路1號 |
| 李大華 | 高雄市左營區XX路2號 |

---

## 第二步：在疏文中設定「變數標籤」

1. 開啟「道教疏文排版系統 (網頁版)」。
2. 點擊您想要放置名單資料的文字區塊開始「編輯」。
3. 在想要讓資料自動帶入的地方，加上 **雙大括號 `{{ }}`**，並輸入您在 Excel 中設定的「標題名稱」。

> **排版範例**：
```
居住於 {{地址}}
信士 {{姓名}} 
```
當系統讀到雙括號 `{{姓名}}` 時，就會知道這裡是要自動替換信眾名字的地方。

---

## 第三步：載入清單並預覽

1. 在螢幕右下方的「工具箱」面板中，找到 **💎 高級版：批次自動套印** 區塊。
2. 點擊綠色的 **「載入 Excel (CSV) 名單」** 按鈕，選擇您剛剛存好的 CSV 檔案。
3. 載入成功後，面板會顯示：「已載入 N 筆資料」。
4. 網頁中央的預覽畫面，會**自動把「第一筆資料」套進去**，讓您檢查字體大小與位置有沒有跑掉。
5. 您可以點擊面板上的「下一筆 / 上一筆」來檢查不同長度的名字或地址是否都能完美塞進去！

---

## 第四步：批次列印與匯出高畫質 PDF

當您確認預覽的位置跟字體大小都很完美後：
1. 點擊右下角最底下的藍色按鈕：**「列印疏文 (N筆)」**。
2. 系統會自動花一點點時間，把這幾百份疏文全部產生出來，並進入列印預覽燈箱。
3. 您可以選擇：
   * **開啟列印**：直接呼叫印表機，開始連續印出所有的紙張。
   * **匯出 PDF**：將所有的疏文打包成一份高畫質（向量等級）的 PDF 檔。這份檔案可以直接傳給專業印刷廠大量印製，保證線條清晰銳利！
